## Webmotors Busca

<hr>

### Dependências
- NodeJs 14: https://nodejs.org/en/

### Instalar as dependências
```bash
$ npm i
```

### Executar o Projeto
- Modo de desenvolvimento:
```bash
$ npm run start
```

- Acessar a url: http://localhost:5000

### Gerar o bundler para produção
```bash
$ npm run build
```
